## ✨ 动画技术

- **预制动画**：打字机、淡入、缩放、平移、旋转、弹跳
- **自定义关键帧①**（P2封面）：标题缩放渐显 + 插画平移入场 + 边框旋转淡入
- **自定义关键帧②**（P5工艺）：糖稀滴落 + 糖丝线条延伸 + 成品图案缩放成型

## 🛠️ 技术栈

纯原生 HTML/CSS/JS，零依赖，单文件即可运行。

- CSS custom properties 配色系统
- CSS animations / keyframes 动画引擎
- Canvas API 糖画模拟
- Touch events 移动端滑动导航
- 响应式缩放适配所有屏幕
